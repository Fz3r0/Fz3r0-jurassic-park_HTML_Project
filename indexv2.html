Paragraph_2 Section_1:
HTML (HyperText Markup Language) es el lenguaje de marcado estándar para la creación de páginas web.
Fue desarrollado por Tim Berners-Lee, un científico de la computación, en 1991 mientras trabajaba en el CERN.

Paragraph_4 (Section_1):
HTML permite estructurar el contenido en la web mediante el uso de etiquetas que definen elementos como encabezados, párrafos, enlaces e imágenes.
Las versiones de HTML han evolucionado con el tiempo, desde HTML 1.0 hasta la última versión, HTML5, que incluye nuevas características como video y audio integrados, y soporte mejorado para aplicaciones web.

Paragraph_5 (Section_1):
Un poco de historia y curiosidades de HTML son que la primera versión, HTML 1.0, se publicó en 1991 y carecía de muchas de las capacidades avanzadas que vemos hoy.

HTML 4.01, lanzado en 1999, introdujo una mayor separación entre el contenido y la presentación, mientras que HTML5, lanzado en 2014, incorporó numerosos elementos y atributos nuevos, incluyendo soporte nativo para gráficos y multimedia sin necesidad de complementos externos.
